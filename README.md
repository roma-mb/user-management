## Getting Started

To get a local copy up and running, follow these simple example steps.

### Prerequisites

* Bower v1.8.14
  ```sh
    npm install -g bower
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:roma-mb/user-management.git
   ```
2. Install bower packages
   ```sh
   # installs the project dependencies listed in bower.json
   bower install
   ```
